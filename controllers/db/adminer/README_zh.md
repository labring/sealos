## sealos cloud adminer 使用文档

### 产品介绍

sealos cloud adminer 是一个基于 kubernetes adminer 的数据库管理工具，可以在 kubernetes 集群中一键部署一个数据库管理服务，
用户可以通过浏览器访问该服务，然后在浏览器中打开一个APP，通过APP可以直接访问 kubernetes 集群内的数据库。
