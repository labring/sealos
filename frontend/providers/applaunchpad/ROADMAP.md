# ROADMAP.md

## Current Focus

- Keep create/edit networking behavior consistent across UI, YAML conversion, v1 APIs, and v2alpha
  APIs.
- Make infrastructure failures understandable to application users.
- Keep API schemas and OpenAPI documentation aligned with actual mutation behavior.

## Near-Term Priorities

- Harden public-domain validation and conflict feedback.
- Maintain image-derived port defaults and route-rule round trips.
- Improve focused regression coverage for schema and networking helpers.
- Keep operational docs current as API behavior changes.

## Non-Goals

- Rebranding internal resource names.
- Replacing the existing Next.js/Chakra stack.
- Broad visual redesigns while fixing narrow networking or schema behavior.
