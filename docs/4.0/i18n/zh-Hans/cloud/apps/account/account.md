## 2、扣费介绍

用户使用资源（比如cpu，memory），就需要在账户中扣除金额

**价格表:**

| 资源名字 | 单位 | 价格（每小时） |
| -------- | ---- | -------------- |
| cpu      | 1核  | 0.067元        |
| memory   | 1Gi  | 0.033元        |
| disk     | 1Gi  | 0.0021         |

如果用户欠费，会影响正常使用，如果欠费太多，则会自动清除用户资源。
